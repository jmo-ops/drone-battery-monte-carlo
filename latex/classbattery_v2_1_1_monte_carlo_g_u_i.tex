\doxysubsection{Detailed Description}
\begin{DoxyVerb}@brief Tkinter GUI for running nominal/extreme Monte Carlo simulations.

Provides:
  - "Simulation" tab: nominal Monte Carlo + histogram plot
  - "Extreme Conditions" tab: extreme-biased Monte Carlo + histogram plot
  - "Parameters Summary" tab: displays configured ranges + battery parameters
\end{DoxyVerb}

Executes the standard Monte Carlo simulation. 

Performs the following process:\+
\begin{DoxyEnumerate}
\item Validates user input for the number of runs.
\item Calls run\+\_\+monte\+\_\+carlo() to perform stochastic trials for each battery.

\item Calculates energy usage metrics and delivery throughput.
